In project one our objective is to create a website using Bootstrap. Setting the website and Webstorm to work with Gulp and BrowserSync to be able to update it live as we make changes. The first thing we do is to install Node.js in order for us to use Gulp and other features to allow us to do live updates to the website. Second, we set-up a Gulp serve for the live website updates. Third, we clone the Bootstrap github page and work from there. Finally, we set-up github to make our website work thru github's "master branch/docs folder".
